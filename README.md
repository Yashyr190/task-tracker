# Personal Task Tracker

## Description
A responsive and user-friendly personal task management application built with React. It allows users to log in, manage their tasks (add/edit/delete/mark complete), and filter them by status â€” all while saving data locally.

## Features
- Simple login using localStorage
- Add, edit, delete tasks with title and optional description
- Toggle task completion status
- Filter tasks by All, Completed, Pending
- Display creation date/time
- Persistent data using localStorage
- Responsive design
